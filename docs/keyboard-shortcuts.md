## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `?` | Open Keyboard Shortcuts reference |
| `Escape` | Close any open modal or panel |
| `Tab` / `Shift+Tab` | Navigate between focusable elements (all modals have focus traps) |

---

## Voice Library — Carousel View

| Shortcut | Action |
|----------|--------|
| `←` Arrow | Move to previous voice |
| `→` Arrow | Move to next voice |
| `Enter` or `Space` | Play voice sample for focused card |
| `F` | Toggle favorite on focused voice |

---

## Script Reader

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Generate speech (when script area is focused) |
| `Space` | Play / pause generated audio |
| `Cmd/Ctrl + S` | Save current script to history |
| `Escape` | Close Script Reader modal |

---

## Review Mode

All shortcuts are active when Review Mode is open and focused.

| Shortcut | Action |
|----------|--------|
| `Space` | Play / Pause current take |
| `R` | Replay (restart from beginning) |
| `A` | Approve current take |
| `F` | Flag current take |
| `N` | Next segment |
| `P` | Previous segment |
| `M` | Add QC Marker (opens issue dialog) |

---

## Command Palette

Open with `Cmd/Ctrl + K`. Type to search for actions:

| Command | Action |
|---------|--------|
| `New Project` | Create a new script project |
| `Open Script Reader` | Open the Script Reader modal |
| `Find My Voice` | Open AI Casting Director |
| `History` | Open generation history panel |
| `Settings` | Open Settings modal |
| `Keyboard Shortcuts` | Show this reference |
| `Toggle Dark Mode` | Switch between light and dark theme |
| `Clear Cache` | Clear the audio cache |

---

## Modals

All modals follow these conventions:

| Shortcut | Action |
|----------|--------|
| `Escape` | Close the modal |
| `Tab` | Move to next focusable element (cycles within the modal) |
| `Shift + Tab` | Move to previous focusable element |
| `Enter` | Activate focused button or submit focused form |

---

## Mini Player

The Mini Player appears at the bottom of the screen during audio playback.

| Shortcut | Action |
|----------|--------|
| `Space` | Play / Pause (when Mini Player is focused) |
| `←` | Seek backward 5 seconds |
| `→` | Seek forward 5 seconds |

---

## Tip

Open the **Keyboard Shortcuts Modal** at any time by pressing `?` or by clicking **Keyboard Shortcuts** in the Command Palette.
